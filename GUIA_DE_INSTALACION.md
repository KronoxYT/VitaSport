# Guía de Instalación - VitaSport

## Requisitos del Sistema
- Windows 10/11 (64-bit)
- 4GB de RAM (8GB recomendado)
- 500MB de espacio en disco
- Conexión a Internet (solo para la primera instalación)

## Instrucciones de Instalación

### Opción 1: Instalación con Asistente (Recomendado)
1. Localice el archivo `VitaSport_1.0.0_x64-setup.exe` en esta carpeta.
2. Haga doble clic para iniciar el asistente de instalación.
3. Siga las instrucciones en pantalla.
   - Acepte los términos de la licencia.
   - Seleccione la carpeta de instalación (por defecto: `C:\Program Files\VitaSport`).
   - Elija si desea crear accesos directos.
4. Haga clic en "Instalar" y espere a que finalice.
5. Una vez completado, haga clic en "Finalizar".

### Opción 2: Instalación Silenciosa (Para Administradores)
1. Abra una ventana de CMD o PowerShell como administrador.
2. Navegue hasta la carpeta que contiene el instalador:
   ```
   cd "ruta\a\la\carpeta"
   ```
3. Ejecute el siguiente comando para una instalación silenciosa:
   ```
   VitaSport_1.0.0_x64-setup.exe /S
   ```

### Opción 3: Paquete MSI (Para Despliegue Empresarial)
1. Localice el archivo `VitaSport_1.0.0_x64_en-US.msi`.
2. Haga doble clic para iniciar la instalación.
3. Siga las instrucciones en pantalla o utilice su herramienta de gestión de software empresarial.

## Primeros Pasos
1. Busque "VitaSport" en el menú de inicio o haga clic en el acceso directo del escritorio.
2. La aplicación se iniciará automáticamente después de la instalación.

## Solución de Problemas
- **Error de permisos**: Asegúrese de tener permisos de administrador para instalar software.
- **Faltan dependencias**: Si la aplicación no inicia, instale los paquetes de Visual C++ Redistributable más recientes.
- **Problemas de rendimiento**: Cierre otras aplicaciones que consuman muchos recursos.

## Desinstalación
1. Abra el Panel de Control > Programas > Desinstalar un programa.
2. Busque "VitaSport" en la lista.
3. Haga clic en "Desinstalar" y siga las instrucciones.
